# 🌤️ Weather Diary - Дневник погоды

**Автор:** Я

## 📝 Описание программы

Weather Diary — это графическое приложение для ведения дневника наблюдений за погодой. Программа позволяет:

- ✅ Добавлять записи о погоде (дата, температура, описание, осадки)
- 🔍 Фильтровать записи по дате и температуре
- 💾 Сохранять данные в JSON-файл
- 🗑️ Удалять ненужные записи
- 📊 Просматривать все записи в удобной таблице

## 🚀 Как запустить приложение

### Требования:
- Python 3.6 или выше
- Tkinter (входит в стандартную установку Python)

### Запуск:

1. Скачайте файл `weather_diary.py`
2. Откройте терминал/командную строку
3. Перейдите в папку с файлом
4. Выполните команду:
```bash
python weather_diary.py
